Analytics

1. Analytics.google.com -> Create new property (Admin / Add new property)
2. Add analytics ID to template
3. Rememeber to include script tag

Tag manager:

1. tagmanager.google.com
2. Add new tag -> Analytics
3. Add variable with analytics ID (override option)
4. Publish (top right)
NB: Remember to update hardcoded gtag id in the <script> tag!
